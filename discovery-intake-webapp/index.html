How to use this app

Run the interview, then package the handoff

  1. Start TalkingAnswer the current question naturally. Speak one answer at a time.
  2. Stop TalkingThe AI transcribes, thinks silently, summarizes what it heard, and asks you to confirm.
  3. Confirm or correctSay yes if the summary is right, or explain what to change.
  4. Add AttachmentUpload screenshots, PDFs, docs, decks, or trackers only when they help clarify the workflow.
  5. Analysis StudioReview the structured handoff, process matrix, Product questions, and Engineering questions.
